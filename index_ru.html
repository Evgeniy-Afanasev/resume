    Учебный проект на базе OpenCV

    Задача: У одного крупного производителя фанеры при доставке бревен для обработки с востока в центральную часть России стали пропадать бревна. Нужно было простое решение, которое упростило бы задачу подсчета количества древесины.

    Решение: Мое решение заключалось в пересчете бревен с помощью обработки изображения на базе библиотеки OpenCV. Небольшой скрипт на Python и вы можете увидеть результат обработки изображения на картинке.
